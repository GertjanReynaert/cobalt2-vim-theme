Cobalt2 Vim
===========

Install
-------

With [Vundle](https://github.com/gmarik/vundle):

    Bundle 'gertjanreynaert/cobalt2-vim-theme'

Contributions
-------------

Pr's are always welcome. Please include a screenshot and a test file that
demonstrates all the highlighted elements
